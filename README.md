rest-api-twitter
================

# forma de uso

solo debes colocar tu cuenta de twitter en en atributo query

    Twitter.init({
    	template: $('#tweets-template').html(),
  		container: $('ul.tweets'),
  		query: 'XXXXXXXXX'
  	 });
